# Hosseyn’s Dotfiles 🛠️

My personal dotfiles, managed by [chezmoi](https://www.chezmoi.io/).

## Usage commands

Install:

```sh
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply hosseynjf
```

Update from GitHub:

```sh
chezmoi update
```
